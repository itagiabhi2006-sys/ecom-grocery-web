# Kirani Store - Frontend v3

What's new in v3:
- Centered Sanātani-themed notification panel (decorative saffron accents, mandala watermark)
- Rich homepage (hero, categories, featured products, testimonials, centered notifications)
- Checkout with address and phone fields (must be logged in to place order)
- Amazon-style footer with multiple columns
- AI-style SVG images for categories and hero (culturally inspired, non-religious visuals)

Run:
1. npm install
2. npm run dev

Replace SVGs in src/assets with real images if needed.
